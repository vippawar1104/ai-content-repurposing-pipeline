# AI Content Repurposing Pipeline

Automatically transcribes audio/video and generates social media content.

## Tech Stack
- Make.com (workflow automation)
- Google Drive (file trigger)
- AssemblyAI (audio transcription)
- Claude Haiku API (content generation)
- Google Docs (output storage)

## How it works
1. Upload audio/video to Google Drive
2. Make downloads and sends file to AssemblyAI
3. AssemblyAI transcribes the audio
4. Claude generates LinkedIn post + Twitter thread + Email newsletter
5. Output saved automatically to Google Docs
